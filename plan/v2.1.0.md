# SuperTabBar Package Update Requirements

Update the Flutter package located at `flutter/super_tab_bar` to version `2.1.0`.

## Keyboard Shortcuts

* Remove the current keyboard shortcuts used for navigating between tabs.

## Compact Mode

* Add support for `compact mode`.
* In compact mode, hide the tab bar.
* In compact mode, provide a screen or view that displays thumbnail previews of all currently open tab pages.
* When the user taps a thumbnail preview, navigate directly to the corresponding tab.
* Allow users to reorder tabs by dragging the thumbnail previews displayed in compact mode.
* Compact mode is intended for small screens, such as mobile phones, where users need an easy way to switch between tabs without displaying the full tab bar.

## Compact Mode Example

* In the example demonstrating compact mode, use a `FloatingActionButton` to open the screen or view that displays thumbnail previews of the open tabs.

## Back Navigation Behavior

* When the user navigates back, close the current tab if it is not marked as `dirty`.
* If the current tab is marked as `dirty`, do not close it automatically when navigating back.

## Documentation

* Update the package documentation according to pub.dev package documentation style and best practices.
* Update `README.md`.
* Update `CHANGELOG.md`.
* Update the AI skill documentation.
* Add a clear example for each newly added feature.

## Expected Outcome

* The package should support a mobile-friendly compact tab navigation experience.
* The package should allow tab reordering through draggable thumbnail previews.
* The package should correctly handle back navigation by closing only tabs that are not marked as `dirty`.
* The package should remove the existing tab-navigation keyboard shortcuts.
* The package should include complete documentation and examples for all new functionality.
