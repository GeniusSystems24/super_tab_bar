# SuperTabBar Package Update Requirements

Update the Flutter package located at `flutter/super_tab_bar` to version `2.5.0`.

## Action Buttons

* If an action is not provided, its corresponding action button must not be displayed in the tab bar. For example, if `onAddTab` is `null`, the Add button must not be shown in the tab bar. The same behavior should apply to all other action buttons.

## BrowserTab

* Rename `BrowserTab` to `SuperTab`.
* Page construction must be handled through a `pageBuilder` function defined within each `SuperTab`, as shown below:

  `SuperTab(pageBuilder: (context) => MyPage())`

  This should replace the shared `pageBuilder` function currently defined in `SuperTabBar`, such as:

  `SuperTabBar(pageBuilder: ...)`

## Documentation

* Update the package documentation according to pub.dev documentation standards and best practices.
* Update `README.md`.
* Update `CHANGELOG.md`.
* Update the AI skill documentation.
* Update all example screens.
